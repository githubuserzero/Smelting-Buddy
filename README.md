[h3]Features[/h3]
[list]
[*]100% automation of your smelting operation! Select ingot, choose amount > press start
[*]Batch smelting operation! Select which ingots and amounts you need and the script will go through each of it without any need for assist
[*]Vent automation to keep autoignition away from your stuff
[*]Included are the obvious status text additions like temperature and pressure
[*]Easy setup through dropdowns in the settings
[*]NO code changes necessary
[/list]

[h2]How to use[/h2]
Just take a quick look at the video. It shows you everything you need to know including a showcase of the batch smelting operation.
You can also use it together with the [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3693742595]Crafting Buddy[/url].

[h2]Setup[/h2]
[list]
[*] Disable Code limits in the IC10 Editor Mod settings
[*] If the code doesn't load in game it may be due to steam workshop. You can check the lines in the code / it should be 3071 lines of code. If Steam workshop doesn't give you the full code you can get it on my GitHub: [url=https://github.com/githubuserzero/Smelting-Buddy/blob/main/Lua%20-%20Smelting%20Buddy.lua]Smelting Buddy[/url]
[/list]
